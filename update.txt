git add .
git commit -m "Atualização: [descreva a mudança]"
git push

git status -->  Verifica o que mudou.

Para remover um arquivo antigo utilize:
git rm nome-do-arquivo
git commit -m "removendo arquivo antigo"
git push


Atualizar repositorio para o novo nome
git remote -v
git remote set-url origin https://github.com/usuario-novo/repositorio.git
